Binary Downloads

Precompiled binaries are available for OS X and Windows and are made available in self-contained packages which should not require any additional libraries or configuration.

  • OS X dmg image.
  • Windows installer or zip file. Links to older versions can be found below.

Source

The latest source code can be obtained from GitHub. For details on obtaining and compiling the code, please see the instructions in the README.md file for further information on compiling from source.

Older Versions

The following past version of IQmol are available, but have been superceded by the latest version and are no longer supported:

  • Windows: 3.1.5   3.1.4   3.1.3   3.1.2   3.1.1   3.0.1   2.15.2  
  • OS X: 3.1.5   3.1.4   3.1.3   3.1.2   3.1.1   3.0.1   2.15.2  
  • Linux (2.14 only): Ubuntu, Fedora or CentOS 7.
